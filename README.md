# Instalacja
* [Zainstaluj VirtualBox](https://www.virtualbox.org/wiki/Downloads)
* [Zainstaluj Vagrant](https://www.vagrantup.com/downloads.html)
* sklonuj repozytorium
* `vagrant up`
* `vagrant ssh`
* `cd /vagrant`
